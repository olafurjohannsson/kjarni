Why Kjarni?

Built for developers who want AI that just works

Self-Contained

Like SQLite for databases, Kjarni is a single library with no external dependencies. No Python runtime, no CUDA toolkit, no containers.

  • Single native library
  • Models load from local files
  • Works offline, air-gapped

Optimized Runtime

Hand-tuned Rust core with SIMD acceleration. Runs on CPU with optional GPU via WebGPU.

  • 4-bit & 8-bit quantization
  • Speculative decoding
  • KV cache optimization

Native Bindings

First-class FFI bindings that feel native in every language. No subprocess overhead.

  • C# / .NET
  • Python
  • Rust (native)
  • Go, Java, C++ (planned)

# Interactive chat with Llama
$ kjarni chat --model llama-3.2-3b-instruct

# With system prompt
$ kjarni chat -m qwen2.5-3b -s "You are a helpful assistant"
Classification
# Classify sentiment
$ echo "I love this product" | kjarni classify

# Custom labels
$ kjarni classify "Server is down" --labels "urgent,normal,low"
Translation
# Translate to Icelandic
$ kjarni translate -i "Hello world" --dst is

# Specify source and target
$ kjarni translate -i "Bonjour" --src fr --dst en
Transcription
# Transcribe audio file
$ kjarni transcribe meeting.mp3

# With timestamps
$ kjarni transcribe podcast.wav --timestamps
Search & Index
# Create an index from a folder
$ kjarni index create ./docs --name my-docs

# Search the index
$ kjarni search ./my-docs "password reset" --top-k 5
Reranking
# Rerank documents by relevance
$ kjarni rerank "best programming language" \

    "Rust is fast" \

Chat
use kjarni::chat::{Chat, ChatConfig};

let config = ChatConfig::default()
    .model("llama-3.2-3b-instruct")
    .system("You are helpful.");

let mut chat = Chat::new(config)?;

for token in chat.stream("What is Rust?")? {
    print!("{}", token);
}

Reranking
use kjarni::rerank::Reranker;

let reranker = Reranker::new("bge-reranker-base")?;

let docs = vec!["doc1", "doc2", "doc3"];
let ranked = reranker.rerank("query", &docs, 3)?;
